#FILE
- file img_to_array.py berisi code untuk
menyimpan nilai piksel citra menjadi numpy dan
code untuk deteksi corner
- file naive_bayes.py berisi code untuk naive bayes classifier
- file app.py berisi tampilan gui menggunakan flask
dan proses training serta test
- file data test (diluar dataset) terdapat dalam folder
static>assets>data_test (untuk mencoba perlu megaktifkan gui)

#FOLDER
- folder dataset berisi dataset keseluruhan,
terdapat 4 sub folder msg-msg berisi 50 citra
- folder laporan berisikan laporan uas
- folder numpy berisi hasil konversi numpy citra 
dari file img_to_array.py
- folder static berisi tampilan css gui dan data test
- folder templates berisi file html untuk gui
